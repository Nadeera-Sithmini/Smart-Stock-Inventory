# 📦 Smart Stock Inventory System

A modern web-based Stock Inventory Management System developed using Java, JSP, Servlets, and MySQL. This application helps businesses efficiently manage inventory, monitor stock levels, and generate reports through an easy-to-use dashboard.

---

## 🚀 Features

- 🔐 Secure Admin Login
- 📦 Add New Products
- ✏️ Update Product Details
- ❌ Delete Products
- 📊 Stock Dashboard
- 🔍 Search Products
- 📈 Stock Reports
- ⚠️ Low Stock Monitoring
- 📅 Inventory Tracking

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Nadeera-Sithmini/Smart-Stock-Inventory.git
```

2. Open the project in NetBeans.

3. Import the MySQL database.

4. Configure database connection.

5. Run the project using Apache Tomcat.
