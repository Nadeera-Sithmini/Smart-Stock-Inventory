# 📦 Smart Stock Inventory Management System

An AI-Powered Smart Stock Inventory Management System developed using Java, JSP, Servlets, and MySQL. The system provides inventory management, POS billing, sales tracking, and predictive AI stock forecasting with real-time analytics.

---

## 🚀 Features

### 🔐 Secure Authentication
- User Login System
- Session Management
- Secure Access Control

### 📦 Inventory Management
- Add New Products
- Update Product Details
- Delete Products
- Search Products
- Live Inventory Ledger
- Real-time Stock Monitoring

### 🛒 Sales & Billing System
- Create New Invoices
- Sell Products Using Product ID
- Automatic Stock Deduction
- Print Bills / Receipts
- Recent Sales History
- Revenue Tracking

### 🤖 AI Reports & Insights
- Predictive Stock Run-Out Forecasting
- Average Daily Sales Analysis
- Estimated Remaining Days
- AI Critical Reorder Alerts
- Total Asset Value Calculation
- Smart Inventory Recommendations

### 📊 Dashboard Analytics
- Total Stock Assets Value
- Total Item Quantity
- Critical Reorder Notifications
- Real-time Business Insights

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Nadeera-Sithmini/Smart-Stock-Inventory.git
```

2. Open the project in NetBeans IDE.

3. Configure Apache Tomcat Server.

4. Create a MySQL database.

5. Import the SQL file.

6. Configure database connection in:

```java
DBConnection.java
```

7. Run the project using Apache Tomcat.
